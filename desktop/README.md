# 🖥️ NN+ Вызов — десктоп-клиент (Windows)

Тонкая нативная обёртка над веб-приложением на **PyQt6 (QtWebEngine)**. Открывает сайт в отдельном окне и добавляет то, чего не может PWA: системный трей, всплытие поверх окон при вызове, автозапуск и защиту от случайного закрытия.

## Возможности

- 🖥️ Отдельное окно приложения (не вкладка браузера).
- 📥 **Крестик сворачивает в трей**, а не закрывает — вызовы продолжают приходить.
- ⬆️ **Всплытие поверх всех окон** при входящем вызове/сообщении (сайт ставит спецсимвол в `document.title`, обёртка реагирует).
- 🚀 Автозапуск при входе в Windows (реестр `Run`), переключается из трея.
- 🔗 Ярлык на рабочем столе одним кликом, с фирменной иконкой.
- 🎨 Анти-мерцание: отключение GPU-композитинга + инъекция CSS, гасящая тяжёлый `backdrop-filter` и фоновые анимации (только в обёртке).

## Файлы

| Файл | Назначение |
|------|-----------|
| `nn_app.py` | Само приложение (окно, трей, ярлык, автозапуск) |
| `make_ico.py` | Генерация иконки `app.ico` через Pillow |
| `NN_Vyzov.spec` | Конфиг сборки PyInstaller (onedir) |
| `app.ico` | Фирменная иконка NN+ |

## Сборка

```bash
pip install PyQt6 PyQt6-WebEngine pyinstaller pillow

python make_ico.py            # сгенерировать app.ico
pyinstaller NN_Vyzov.spec      # → dist/NN_Vyzov/NN_Vyzov.exe
```

## Настройка

URL сервера берётся из переменной окружения `NN_URL`
(по умолчанию — заглушка `https://your-server.example.com`):

```bash
set NN_URL=https://ваш-домен        # Windows
```

> Запуск: `python nn_app.py` (для разработки) или собранный `NN_Vyzov.exe`.
